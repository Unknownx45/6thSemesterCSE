1. Python Identifiers and Variables

Identifier: A name used to identify variables, functions, classes, modules, etc.

Rules for identifiers:

  • Must start with a letter (a-z, A-Z) or underscore (_)
  • Can contain letters, digits, and underscores
  • Case-sensitive
  • Cannot be a Python keyword

Example:

# Valid identifiers
name = "Python"
_private = 10
count1 = 5

# Invalid identifiers
# 1st = 10  # starts with digit
# class = "OOP"  # Python keyword

2. Data Types

Python has several built-in data types:

Type Description Example Function
int Integer numbers 10, -5, 0 int()
float Floating point numbers 3.14, -2.5 float()
str String of characters "Hello", 'Python' str()
bool Boolean values True, False bool()
complex Complex numbers 3+4j, 2-1j complex()

3. Input and Output Operations

Output using print()

Example:

print("Hello, World!")
print("Name:", "Python", "Version:", 3.9)
print(f"The answer is {42}")

# Output:
# Hello, World!
# Name: Python Version: 3.9
# The answer is 42

Input using input()

Example:

name = input("Enter your name: ")
age = int(input("Enter your age: "))
print(f"Hello {name}, you are {age} years old")

4. Operators

Python supports various types of operators:

Type Operators Example Result
Arithmetic +, -, *, /, //, %, ** 10 // 3 3
Comparison ==, !=, <, >, <=, >= 5 > 3 True
Logical and, or, not True and False False
Assignment =, +=, -=, *=, /= x += 5 x = x + 5

5. Type Conversions

Python supports both implicit and explicit type conversions:

Explicit Type Conversion:

# String to int
num_str = "123"
num_int = int(num_str)  # 123

# Float to int
num_float = 3.14
num_int = int(num_float)  # 3

# Int to string
age = 25
age_str = str(age)  # "25"
